# Tutor LMS Courses Carousel Widget

Widget personalizado para **Elementor** que exibe **cursos do Tutor LMS** em um **carrossel responsivo**.

Projeto criado para uso em **projetos específicos WordPress + Tutor LMS**.

## Tecnologias

- WordPress Plugin API  
- Elementor Widget API  
- Tutor LMS  
- Swiper.js
